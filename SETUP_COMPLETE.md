## 🚀 如何使用

### 克隆仓库

```bash
git clone https://github.com/Tonystarkw12/pomapython.git
cd pomapython
```

### 快速开始

```bash
# 1. 激活 conda 环境
conda activate wolfram

# 2. 运行快速 REPL
python quick_run.py

# 或运行详细演示
python run_poma_interactive.py
```
